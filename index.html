R·01 — Indicium AI/ML Program · Feb 2026 – present
LLM pipeline for clinical data labelling
Building an automated pipeline for clinical data labeling from medical notes using large language models — replacing slow, error-prone manual labelling with something reliable enough for real medical datasets.
Sole undergraduate Python developer on the team. Designed the prompt framework, evaluation methodology, and iterated through dataset issues suppressing accuracy. Improved label accuracy from 45% → 65% — the gap came mostly from fixing how we measured, not just the model.
Co-authoring a research thesis and conference poster. Submitting to a national conference in July 2026.

P·01 — Ergonomic Adjustable Mouse · Dec 2025 – present
Ergonomic adjustable mouse project
Full hardware project from scratch — including schematic design, custom PCB layout in KiCad, Arduino-based C++ firmware, sensor debugging, and 3D-printed enclosure assembly.
Integrated ADNS-3080 optical sensor. Hit signal instability early — traced to noise on the SPI lines, fixed with hardware filtering and firmware-level debounce. Currently transitioning from working breadboard prototype to a manufacturable board revision.
